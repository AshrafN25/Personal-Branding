# Instruksi Menambahkan Gambar

## 1. Gambar Project Era Reformasi

### Langkah-langkah:

1. **Ambil Screenshot dari Website Era Reformasi**
   - Buka website: https://era-reformasi.vercel.app
   - Ambil screenshot dari halaman utama (hero section dengan gambar monumen)
   - Pastikan screenshot menampilkan bagian terbaik dari website

2. **Simpan Gambar**
   - Simpan screenshot dengan nama: `era-reformasi.jpg`
   - Letakkan file di folder: `public/`
   - Path lengkap: `c:\Users\NITRO V\Downloads\pop\ashraf-portfolio\public\era-reformasi.jpg`

3. **Format Gambar**
   - Format: JPG atau PNG
   - Ukuran yang disarankan: Lebar minimal 800px
   - Rasio aspek: 16:9 atau 4:3

---

## 2. Gambar Sertifikat Coding Camp 2026

### Langkah-langkah:

1. **Simpan Gambar Sertifikat**
   - Gunakan gambar sertifikat yang Anda miliki
   - Simpan dengan nama: `Coding Camp 2026 DBS Foundation.jpg`
   - Letakkan file di folder: `public/certificates/`
   - Path lengkap: `c:\Users\NITRO V\Downloads\pop\ashraf-portfolio\public\certificates\Coding Camp 2026 DBS Foundation.jpg`

2. **Format Gambar**
   - Format: JPG atau PNG
   - Ukuran: Sesuai dengan sertifikat asli
   - Pastikan teks pada sertifikat dapat terbaca dengan jelas

---

## Verifikasi:

Setelah menambahkan kedua gambar:

1. **Project Era Reformasi**: 
   - Buka browser dan refresh
   - Cek di section Portfolio → Projects
   - Project "Era Reformasi" seharusnya muncul di posisi ke-3

2. **Sertifikat Coding Camp**:
   - Buka browser dan refresh
   - Cek di section Portfolio → Certificates
   - Sertifikat "Coding Camp 2026" seharusnya muncul di posisi pertama (paling atas)

---

## Catatan:
- Pastikan nama file **PERSIS SAMA** dengan yang tertulis di instruksi
- Jika nama file berbeda, gambar tidak akan muncul
- Ekstensi file (.jpg atau .png) harus sesuai dengan nama file yang digunakan
